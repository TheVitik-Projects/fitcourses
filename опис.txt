Сайт з умовною назвою fitcourses призначений для студентів і викладачів для організації навчання в інтернеті.
Функціонал для викладача: 
Курс:
-створення курсу з можливістю додавати завдання;
-перегляд створених завдань та їх вміст (текст, дата, вкладені файли, посилання);
-перегляд учасників курсу та їх даних
-можливість поділитись посиланням на курс
-видалення курсу
Тест: 
-створення тесту
-перегляд усіх тестів
-перегляд результатів проходження тесту
-можливість поділитись посиланням на тест
Функціонал для студента:
-приєднання до курсу
-можливість бачити завдання курсу
-можлиість переглядати прикріплені файли
-проходження тесту та отримання результату

Технології, використані для написання сайту:
Фронтенд:
-HTML
-CSS
-JavaScript
-Bootstrap Framework
-Шрифти FontAwesome
Бекенд:
-PHP
-Бази даних MySQL

Структура сайту:
Коренева директорія сайту має 6 папок та 7 PHP файлів.
Папки: 
1. css - файли стилів
2. files - файли, які завантажені у форму при створенні завдань
3. images - фото, використані на сайті
4. js - JavaScript файли
5. pages - директорія папок з файлами для кожного блоку сторінок.
6. tools - файли PHP, обробники даних з форм + конфігурація підключення до БД MySQL
Файли:
1. account.php - відповідає за контент на сторінці акаунту.
2. course.php - відповідає за контент на сторінці курсу.
3. index.php - головна сторінка сайту
4. signin.php - форма входу в акаунт
5. signup.php - форма реєстрації на сайті
6. teachers.php - сторінка для реєстрації акаунту викладача
7. tests.php - відповідає за контент на сторінці тестів.
